# Cronograma

## Sprints

O projeto será desenvolvido utilizando a metodologia ágil Scrum, com Sprints de 2 semanas, alinhadas ao calendário letivo da disciplina.

| Sprint | Período | Objetivo | Entregas |
| ------ | ------- | -------- | -------- |
| Sprint 1 | 27/03 - 10/04 | Problematização e Planejamento | Relatório de Imersão, Mapa de Empatia, Backlog, MVP |
| Sprint 2 | 10/04 - 24/04 | Engenharia de Dados (ETL) | Scripts de extração e limpeza (Pandas), Diagrama de Modelagem de Dados |
| Sprint 3 | 24/04 - 08/05 | Estruturação da Base | Banco de dados PostgreSQL populado, Documentação Técnica |
| Sprint 4 | 08/05 - 22/05 | Análise e Modelagem | Relatório de Análise Estatística, Treinamento do Modelo Preditivo (Random Forest) |
| Sprint 5 | 22/05 - 05/06 | Visualização e Interface | Scripts de gráficos, Dashboard Interativo (Streamlit) |
| Sprint 6 | 05/06 - 19/06 | Insights e Segurança | Insights, Relatório de Segurança e Privacidade (LGPD) |
| Sprint 7 | 19/06 - 26/06 | Finalização e Entrega | Protótipo, Pitch, Documentação |

## Marcos de Avaliação (Milestones)

- 23/04/2026: Entrega do MVP

- 15/05/2026: Entrega do Relatório Técnico de ETL / Programação com Pandas
- 26/06/2026: Entrega Final
